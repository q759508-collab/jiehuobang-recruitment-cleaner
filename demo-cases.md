# 演示案例

## 案例 1：岗位招聘清洗成 JSON

输入：

```text
使用 jiehuobang-recruitment-cleaner，把下面招聘信息清洗成 JSON：

成都新都食品厂招包装女工，18-45岁，白班早8点30到晚6点，180一天，包住不包吃，张姐 13800000000
```

期望展示：

- 类型：岗位招聘。
- 城市：成都。
- 区域：新都。
- 岗位：食品厂包装女工。
- 年龄：18-45。
- 时间：08:30-18:00。
- 薪资：180/天。
- 联系人：张姐。

## 案例 2：风险检查

输入：

```text
使用 jiehuobang-recruitment-cleaner，检查下面招聘信息有没有风险：

高薪急招普工，月入8000-12000，包吃包住，先交300服装费，安排进厂，电话13800000000
```

期望展示：

- 风险：服装费。
- 风险：高薪夸张。
- 缺失：城市、具体地址、年龄要求、招聘主体。
- 建议：先核实费用、地址、主体。
