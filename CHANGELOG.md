# Changelog

## v0.1.3-alpha

发布后补强版本，重点是让试用者更容易上手，并让项目方能观察真实需求。

### Added

- 新增 `USER_TEST_GUIDE.md`：说明如何拿真实招聘文本试用、如何检查输出、如何反馈。
- 新增 `ADOPTION_METRICS.md`：记录 Star、Fork、Issue、真实样本、批量/Excel/网页/微信采集等需求信号。
- README 增加试用指南、反馈入口和使用情况观察入口。

### Changed

- `VERSION` 更新为 `0.1.3`。
- 发布说明标签更新为 `v0.1.3-alpha`。

## v0.1.2-alpha

第一版公开 alpha。

### Added

- `SKILL.md`：接活帮招聘信息清洗 Skill 入口。
- 招聘、求职、临时活、招人需求、劳务发单的字段和风险规则。
- Word 导出脚本。
- CSV 导出脚本。
- JSON 校验脚本。
- 重复信息检测脚本。
- 脱敏后的真实微信群风格样本。
- 代表性预期输出样本。

### Notes

- 真实业务输出默认保留手机号、微信号和联系人。
- 开源仓库样例使用 `13800000000` 演示，避免公开真实联系人。
- 当前版本不做微信采集，不做数据库后台，不做平台。
