# 试用指南

这个项目先验证一件事：

```text
混乱的微信群招聘文本 -> 清洗成可复制、可整理、可导出的结果
```

## 适合拿来测试的文本

- 工厂招聘。
- 劳务发单。
- 日结兼职。
- 求职者自荐。
- 招人需求。
- 一条消息里多个岗位。

请先删除真实隐私信息再公开反馈。如果是在自己电脑上本地使用，可以保留手机号、微信号和联系人。

## 推荐测试方法

### 1. 复制一条招聘原文

示例：

```text
成都新都包装厂招女工10名，18-48岁，白班，包住，月薪4500-5500，李老师 13800000000
```

### 2. 在 Codex 中输入

```text
使用 jiehuobang-recruitment-cleaner，把下面招聘信息清洗成 JSON：

这里粘贴招聘原文
```

### 3. 看这几项是否正确

- 城市、区域、岗位。
- 年龄范围。
- 薪资。
- 班次和面试时间。
- 联系人、电话、微信。
- 返费、开票扣税、商保、体检费、押金。
- 自离、旷离、卡点、水军、投诉不结费等风险。

### 4. 导出 Word

准备好 JSON 后，可以运行：

```bash
python scripts/export_docx.py samples/representative-expected-output.json -o output.docx
```

Word 更适合复制、发给同事、人工二次修改。

## 怎么反馈

如果识别错了，请按这个格式提交 Issue 或反馈：

```text
【原文】
粘贴脱敏后的招聘信息

【哪里不对】
例如：2553岁没有识别成25-53岁 / 联系电话漏了 / 返费没保留

【希望结果】
说明你希望怎么整理

【使用场景】
微信群整理 / 招聘网站录入 / Word发给同事 / Excel汇总
```

## 暂不支持

- 自动采集微信。
- 数据库后台。
- 支付和会员。
- 自动发布到招聘网站。

这些功能会根据真实试用反馈决定是否继续做。
