# 私有样本目录说明

真实微信群样本不要提交到公开仓库。

建议本地另建私有目录，例如：

```text
C:\Users\Administrator\jiehuobang-private-samples
```

目录结构：

```text
jiehuobang-private-samples/
  raw/
    2026-06-07-wechat-raw.md
  cleaned-json/
    2026-06-07-cleaned.json
  word/
    2026-06-07-output.docx
  feedback/
    trial-feedback-log.md
```

公开仓库只提交：

- 脱敏样本。
- 规则改进。
- 脚本改进。
- 文档改进。

不要提交：

- 真实手机号。
- 真实微信号。
- 身份证号。
- 未经允许的群聊原文。
- 企业内部结算信息。
