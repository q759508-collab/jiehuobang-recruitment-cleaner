# 试用邀请文案

## 可直接公开发布版

我做了一个免费开源的招聘信息清洗 Skill，主要处理微信群、劳务群、招聘群里的混乱文本。

它可以把一条招聘广告整理出：

- 城市、岗位、薪资、年龄、班次。
- 联系人、电话、微信。
- 返费、开票扣税、商保、体检费、押金。
- 卡点离职、自离旷离、水军投诉不结费等风险。
- JSON、Word 文档、表格或正式发布文案。

项目地址：

```text
GitHub: https://github.com/q759508-collab/jiehuobang-recruitment-cleaner
Gitee: https://gitee.com/ffawei/jiehuobang-recruitment-cleaner
```

现在第一版不做微信采集、不做平台、不做后台，只验证“招聘信息清洗”这件事有没有真实需求。

如果你平时要整理招聘群信息，可以拿 3-5 条脱敏后的真实样本试一下，也欢迎提 Issue。

## 反馈引导

你帮我看 4 个点就行：

1. 类型判断准不准。
2. 字段有没有提取错。
3. 风险有没有漏。
4. 你会不会需要批量导出 Excel。

## 样本反馈格式

```text

这里粘贴脱敏后的招聘信息

【哪里不对】
例如：年龄识别错了 / 电话漏了 / 返费没保留 / Word 不方便复制

例如：整理成 Word / Excel / 发布文案 / JSON / 网站录入格式

【使用场景】
```
