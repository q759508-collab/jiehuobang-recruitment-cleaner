# v0.1.3-alpha 发布说明

这是接活帮招聘信息清洗 Skill 的试用增强版。

## 适合谁

- 招聘群群主。
- 县城招聘站运营者。
- 劳务公司。
- HR 服务公司。
- 每天从微信群整理招聘信息的人。

## 这个版本做了什么

- 把混乱的微信群招聘文本整理成 JSON、Word 文档、表格或发布文案。
- 保留联系人、电话、微信、多联系人角色。
- 识别年龄、薪资、班次、面试时间、地址、返费、商保、体检费、押金等字段。
- 提醒卡点离职无费用、自离旷离无工资、开票扣税、水军投诉不结费等风险。
- 新增试用指南，方便外部用户拿真实文本测试。
- 新增使用情况观察指标，帮助判断是否值得继续做批量、Excel、网页或微信采集。

## 当前不做什么

- 不自动采集微信。
- 不保存数据库。
- 不提供后台系统。
- 不做支付和会员。

## 试用方式

在 Codex 中输入：

```text
使用 jiehuobang-recruitment-cleaner，把下面招聘信息清洗成 JSON：

这里粘贴招聘原文
```

如果要导出 Word：

```bash
python scripts/export_docx.py samples/representative-expected-output.json -o output.docx
```

## 反馈重点

请优先反馈：

- 年龄、日期、时间是否识别错。
- 联系方式是否漏提取。
- 多岗位是否拆分正确。
- 返费、商保、体检费、押金、开票扣税是否保留。
- Word 是否方便复制和二次修改。

## 隐私说明

真实业务使用时可以保留手机号、微信号和联系人。公开反馈或提交 Issue 时，请先脱敏真实联系方式。
